# Machine Learning Cross Validation Projects

This repository demonstrates different cross validation techniques used in machine learning using real examples.

Projects Included

1. Cancer Detection System
Algorithm: Random Forest
Cross Validation: Stratified K-Fold

2. Stock Price Prediction
Algorithm: Linear Regression
Cross Validation: Time Series Split

3. Customer Churn Prediction
Algorithm: Decision Tree
Cross Validation: K-Fold

4. Rare Disease Research
Algorithm: Support Vector Machine (SVM)
Cross Validation: Leave-One-Out Cross Validation (LOOCV)

5. Image Classification System
Algorithm: Support Vector Machine (SVM)
Cross Validation: Repeated K-Fold

Technologies Used
Python
Scikit-learn
Jupyter Notebook
Machine Learning

Purpose
This repository shows how different cross validation techniques are applied depending on dataset type such as imbalanced datasets, time series data, small datasets, and large datasets.
